Agri-QA Pipeline v2

基于知识图谱与大语言模型的农业病虫害问答推理链自动生成系统

FAISS 语义检索 结构化 CoT 自洽性验证 MiMo-v2.5-Pro
1 问题与动机

现有农业问答 pipeline 在实体识别、知识检索和推理质量三个环节存在明显短板

实体识别覆盖率低

原系统基于硬编码同义词表（约 200 条）做关键词匹配，无法覆盖 CropDP-KG 中 3,500+ 病虫害实体，遗漏率高。

瓶颈 1

KG 检索准确率不足

基于子串匹配检索知识图谱，无法处理语义相似但字面不同的查询（如 "leaf spots" vs "叶斑病"），召回率低。

瓶颈 2

推理链质量不可控

使用固定 prompt 模板生成推理链，无自洽性检查和事实验证机制，生成结果的准确性和一致性无法保证。

瓶颈 3

4 核心模块详解

每个模块独立可测试，共 31 个单元测试全部通过

4.1 混合实体识别

关键词匹配：从 CropDP-KG-EN 自动构建 2,094 条术语词典，按长度降序匹配，支持多词有序回退（允许词间间隔）。

FAISS 语义补全：question embedding → entity_index top-5，置信度 > 0.7 才采纳。

实体分类：自动识别 crop / pest / disease / weed 类型。

关键词 + 语义混合

4.2 FAISS 向量检索

使用 bge-small-en-v1.5 (384 维) 对 KG 条目编码，构建 IndexFlatIP 索引。检索时 L2 归一化后用内积计算余弦相似度，返回 top-5 最相关 KG 条目。

两个独立索引：kg_index（全量 KG）和 entity_index（实体名称）。

向量检索

4.3 结构化推理链生成

使用 DeepSeek-R1 / MiMo-v2.5-Pro 生成结构化 CoT 推理链。输出 XML 格式，包含 question_analysis、evidence_retrieval、reasoning_steps、conclusion 四个部分。

temperature=0.2 确保输出稳定。

结构化 CoT

4.4 自洽性检查

采样 N=3 次（temperature=0.4），比较诊断结论一致性。2/3 或 3/3 一致为 high_confidence；否则标记 low_confidence 并取首次结果。

若 3 次均解析失败，降低温度重试一次。

多采样投票

4.5 后置验证

将推理链与 KG 证据交叉校验。LLM 逐条检查推理链中的声明是否与 KG 一致，输出 verified / partially_verified / contradicted 状态。

标记矛盾部分和无证据支持的声明。

事实校验

4.6 数据筛选

从 44,615 条 AgThoughts QA 中筛选病虫害子集：Biotic 类别直接纳入 (1,629 条)；Plant and Seed Health 类别用 CropDP-KG 实体名做关键词匹配。

最终筛选出 1,822 条有效样本。

AgThoughts (QA 数据)

英文农业问答数据集，覆盖作物管理、病虫害、土壤、天气等多个领域。本项目聚焦病虫害子集：

  • Biotic Diseases Questions (819 条)
  • Biotic Insects Questions (702 条)
  • Biotic Weeds Questions (108 条)
  • Plant and Seed Health (关键词匹配子集)

CropDP-KG (知识图谱)

中文病虫害知识图谱，每条包含：

  • 名称 + 英文名
  • 为害症状 (symptoms)
  • 发生规律 (occurrence)
  • 防治措施 (prevention)

通过 LLM 批量翻译为英文，构建双语 KG。

7 关键发现与改进方向

测试验证了 pipeline 的可行性，也暴露了需要优化的环节

验证通过

  • FAISS 语义检索有效提升了 KG 召回率
  • 结构化 CoT 输出质量明显优于固定模板
  • 自洽性检查可过滤不稳定输出
  • 中间保存机制防止了数据丢失
  • 1,822 条筛选子集覆盖率合理

待改进

  • 4/10 验证步骤 XML 解析失败（模型输出格式不稳定）
  • 部分实体数为 0（术语词典需扩充作物俗名）
  • 每条 3-4 分钟（自洽性 3 次采样 + 验证 = 4 次 API 调用）
  • KG 翻译仍为中文（LLM 翻译因余额不足跳过）

后续方向

  • 优化验证模块 XML 解析容错（正则 + 宽松解析）
  • 扩充术语词典（作物俗名、区域别名）
  • 完成 KG 全量英文翻译
  • 全量运行 1,822 条，评估整体质量
